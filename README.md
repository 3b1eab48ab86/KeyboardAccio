Keyboard Accio
================
* A tweak makes it easier to switch between keyboards

Features
================
* Tap "Globe" key to switch between first and second keyboards
* Hold "Globe" key to access other keyboards
* Supports 3rd-party keyboards (iOS 8)
* Supports iOS 6 ~ 8
